# 🚀 Real-Time Crypto Analytics Dashboard

## Project Overview
This project is an **end-to-end real-time cryptocurrency analytics system** for tracking **Bitcoin and Ethereum**. It demonstrates the ability to fetch live data, store it efficiently, and visualize it professionally. The dashboard helps analyze **price trends, market capitalization, and trading volume** in real-time.

---

## Features
- **Live Data Integration:** Fetches real-time cryptocurrency data from the CoinGecko API.
- **Data Storage & Management:** Stores data in SQLite database and CSV for historical analysis.
- **Interactive Dashboard (Power BI):**  
  - KPI Cards: Price, Market Cap, Volume  
  - Price & Volume Trend Charts  
  - Coin Selector for filtering  
- **Real-Time Updates:** Dashboard continuously reflects the latest data.

---

## Tech Stack
- **Python:** Data fetching, processing, and storage  
- **SQLite:** Efficient data storage for historical records  
- **Power BI:** Interactive data visualization and dashboard creation  
- **Streamlit (optional):** Web dashboard for live updates  
- **Data Visualization:** Charts, KPI cards, trend analysis
